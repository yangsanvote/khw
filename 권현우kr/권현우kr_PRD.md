# 권현우 양산시의회의원 선거 웹사이트 PRD
작성일: 2025-02-24
버전: 2.0
작성자: Monica
## 개요
본 웹사이트는 2025년 4월 2일 양산시의회 보궐선거 후보 권현우의 공식 선거 웹사이트입니다. 단순하고 명확한 메시지 전달을 위해 원페이지 스크롤 방식으로 구현됩니다.
## 핵심 목표
1. "사는 곳을 바꿉니다, 삶을 바꿉니다" 메시지 강조
2. 후보자의 진정성 전달 ("다르다, 진짜다")
3. 현 시의회의 문제점과 해결책 제시
4. D-Day까지 남은 시간을 효과적으로 전달

## 기술 스택
- Frontend: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion
- UI Components: shadcn/ui
- Deployment: Vercel
- Analytics: Vercel Analytics
  
## 페이지 구조
### Hero Section
- D-Day 카운터
  - 선거일: 2025년 4월 2일 기준
  - 실시간 업데이트
- 메인 슬로건: "사는 곳을 바꿉니다, 삶을 바꿉니다"
- 서브 메시지: "다르다, 진짜다"
- 방문자 카운터
- 프로필 이미지 중앙 배치

### Declaration Section (출마선언문)
- 배경: 밝은 이미지 + 오버레이
- 전문보기 모달 기능
- 주요 메시지 하이라이트
- 날짜 및 서명

### Press Section (언론보도)
- 그라데이션 다크 테마
- 최신순 정렬된 기사 목록
- 각 기사별 링크 연결
- 스크롤 가능한 목록

### Apt Section (아파트 경험)
- 배경: 청어람 아파트 이미지
- 제목: "아파트의 개념을 새로 쓴 사람"
- 주요 정책:
  1. 공동주택 유지보수 지원
  2. 주민갈등 조정위원회 설립
  3. 아파트 종사자 안전 지원

### Care Section (돌봄 정책)
- 배경: 그라데이션 오버레이 이미지
- 제목: "돌봄, 정주, 일자리 세마리 토끼를 한 번에"
- 주요 정책:
  1. 다함께 돌봄센터 유치
  2. 경로당 환경 개선 지원사업 확대

### worker section (자영업자정책)
- 배경: 보라색 그라데이션 (from-[#623D91] via-purple-950 to-[#623D91])
- 제목: "돈은 '잘' 써야합니다"
- 서브타이틀: "일하는 사람들에게 힘이되는 시의원"
- 주요 정책:
  1. 자영업자 이자 5% 지원 조례
     - 진주처럼 자영업자의 이자지원을 5%로 늘리는 정책
  2. 일자리 창출 우수기업 지원
     - 지역 시민 채용기업에게 관급 입찰 우선권 부여

- UI 특징:
  - 블러 효과가 적용된 카드 디자인
  - 반응형 그리드 레이아웃 (모바일 1열, 데스크톱 2열)
  - 아이콘과 함께 제시되는 정책 카드
  - 스크롤 인디케이터 포함
  - Framer Motion을 활용한 애니메이션 효과
    - 제목: 위에서 아래로 페이드인
    - 카드: 좌우에서 중앙으로 슬라이드인
  

### support section (자영업자지원정책도움)
- 배경: 연한 초록색 그라데이션 (from-green-50 to-green-100)
- 제목: "자영업 경력 8년, 삶의 고단함을 아는 후보"
- 서브타이틀: "자영업자 지원사업을 도와드립니다"

- 지원사업 안내 (클릭 시 상세페이지 연결):
  1. 소상공인 경영환경개선 지원
     - 2025년 소상공인 소규모 경영환경개선 사업 및 디지털 인프라 지원
  2. 고용보험료 지원
     - 2025년 소상공인 고용보험료 지원사업
  3. 정책자금 지원
     - 2025년도 경상남도 소상공인 정책자금 운용계획
  4. 고효율기기 지원
     - 2024년 효율향상사업 소상공인 고효율기기 지원
  5. 주방 위생환경 개선
     - 2025년 일반음식점 주방 위생환경개선사업 지원
  6. 청소소독비 지원
     - 2025년 식품위생업소 청소소독비 지원사업

- UI 특징:
  - 반응형 그리드 레이아웃
    - 모바일: 2열
    - 태블릿: 2열
    - 데스크톱: 3열
  - 각 지원사업 카드:
    - 흰색 배경의 그림자 효과
    - 아이콘과 제목 조합
    - 호버 시 그림자 확대 효과
  - 문의 연락처 섹션:
    - 전화번호: 010-4855-5375
    - 클릭 시 전화 연결

## UI/UX 특징
1. 스크롤 인디케이터
   - 각 섹션 하단에 다음 섹션 안내
   - 부드러운 애니메이션 효과
2. 반응형 디자인
   - 모바일: < 640px
   - 태블릿: 640px - 1024px
   - 데스크톱: > 1024px
3. 애니메이션
   - Framer Motion을 활용한 부드러운 진입 애니메이션
   - 스크롤 기반 요소 등장 효과
   - 이미지 슬라이더 자동 재생
4. 컬러 스키마
   - Hero: 화이트 베이스에 컬러 포인트
   - Problem: 다크 테마
   - Solution: 라이트 테마
   - Candidate: 그라데이션 배경
